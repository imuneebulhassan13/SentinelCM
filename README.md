# SentinelCM

Lightweight Open Source SIEM for Centralized Configuration and Integrity Monitoring.

## Features

- Centralized Log Collection
- Configuration Integrity Monitoring
- Real-Time Alerts
- Unified Dashboard
- Windows & Linux Agent
- REST API
- MongoDB Storage
- JWT Authentication

## Tech Stack

Backend:
- FastAPI
- MongoDB
- Python

Frontend:
- React
- Tailwind CSS

Agent:
- Python

Database:
- MongoDB
